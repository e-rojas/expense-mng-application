# expense-mng-application
Expense management application written with TypeScript, React-Native and Redux.

## Features
- [x] Add expenses
- [x] Edit expenses
- [x] Delete expenses
- [x] Display expenses
- [x] Login user
- [x] Logout user
- [x] Signup user
- [x] Display user
- [x] Update user profile

## Installation
npm install
